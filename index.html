Guydle is a Wordle clone by Verve based on fangames of I Wanna Be The Guy, where you guess the final word of a game title taken from delicious-fruit.com.
The list of possible games includes all 1346 titles which end in a 5 letter word using only the English alphabet. Your allowed guesses are limited to the same set as Wordle, plus any word used in an actual game title that wasn't on the list.
You have unlimited guesses.
